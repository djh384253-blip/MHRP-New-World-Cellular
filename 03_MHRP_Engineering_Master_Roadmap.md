# 03_MHRP_Engineering_Master_Roadmap

This roadmap outlines the path from initial prototyping to a global, self-sustaining cellular network.

### Phase 1: The "Frankenstein" Prototype
* Assemble proof-of-concept using a Raspberry Pi 4 wired to an SDR prototyping module[span_2](start_span)[span_2](end_span).
* Code core open-source mobile app dialer interface[span_3](start_span)[span_3](end_span).
* Demonstrate successful voice call routing over unlicensed Sub-GHz frequencies[span_4](start_span)[span_4](end_span).

### Phase 2: Open-Source Release & KiCad Engineering
* Translate working prototype hardware layout into unified, clean schematics using KiCad[span_5](start_span)[span_5](end_span).
* Upload 100% of software source code, mobile app packages, and Gerber hardware design files to GitHub[span_6](start_span)[span_6](end_span).
* Launch public documentation verifying the system's un-trackable nature[span_7](start_span)[span_7](end_span).

### Phase 3: The Initial Production Run (200 Units)
* Seed a strategic footprint funded by community donations and pre-orders[span_8](start_span)[span_8](end_span).
* Deploy 100 Mobile/Keychain Nodes for moving personal perimeters[span_9](start_span)[span_9](end_span).
* Deploy 100 Fixed Home Nodes to establish the long-range mesh backbone[span_10](start_span)[span_10](end_span).

### Phase 4: Pricing Model & Regenerative Production Scale
* Implement a non-profit distribution model where 100% of micro-profits from retail sales fund subsidized hardware kits for strategic geographic gaps[span_11](start_span)[span_11](end_span).
* Scale the network organically through community-driven deployment[span_12](start_span)[span_12](end_span).
