# 03_MHRP_Engineering_Master_Roadmap

This roadmap outlines the path from initial prototyping to a global, self-sustaining cellular network.

## Phase 1: The "Frankenstein" Prototype
* Assemble proof-of-concept using a Raspberry Pi 4 wired to an SDR prototyping module.
* Code core open-source mobile app dialer interface with Flutter.
* Demonstrate successful voice call routing over unlicensed Sub-GHz frequencies.
* Validate the mesh networking concept with local prototype deployment.

## Phase 2: Open-Source Release & KiCad Engineering
* Translate working prototype hardware layout into unified, clean schematics using KiCad.
* Upload 100% of software source code, mobile app packages, and Gerber hardware design files to GitHub.
* Launch public documentation verifying the system's un-trackable nature and privacy guarantees.
* Establish community contribution guidelines and developer documentation.

## Phase 3: The Initial Production Run (200 Units)
* Seed a strategic footprint funded by community donations and pre-orders.
* Deploy 100 Mobile/Keychain Nodes for moving personal perimeters and portable coverage.
* Deploy 100 Fixed Home Nodes to establish the long-range mesh backbone infrastructure.
* Conduct field testing and gather real-world performance data.

## Phase 4: Pricing Model & Regenerative Production Scale
* Implement a non-profit distribution model where 100% of micro-profits from retail sales fund subsidized hardware kits for strategic geographic gaps.
* Scale the network organically through community-driven deployment.
* Establish partnerships with privacy advocates and resilience-focused communities.
